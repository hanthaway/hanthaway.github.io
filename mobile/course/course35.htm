12. Let’s camp

福利：后附中英文对照

Let’s camp. We can go in the van

一起去野营，我们可以坐小货车去

We will have fun

我们会玩得很开心

Where do we want to go?

我们想去哪里呢？

Don’t we have a map

我们没有一个地图

Run to van and get a map, Kim

Kim，跑回小货车去拿地图

This is a mess

这是一团乱

Where is the map

地图在哪里呢

Look I found it

看吧，我找到它了

Now where do we want to go

现在我们要去哪里呢

Let’s go here. This camp has sand

让我们去这里吧，这个营地有沙滩

Get in the van, and let’s go camp

上车，让我们去野营吧
