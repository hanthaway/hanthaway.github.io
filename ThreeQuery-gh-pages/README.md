# ThreeQuery
基于Three.js的3D框架，支持移动设备以及VR效果
## api文档：https://hypnosnova.github.io/ThreeQuery/docs/index.html
<br>
# 核心思想：没必要进行额外的封装，只让使用three.js常用功能变得更加轻松容易。
